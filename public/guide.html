Backstage 사용 가이드

무대에 오르기 전, 묵묵히 합을 맞추는 공간.
밴드 합주를 위한 악보 자동 넘김 서비스입니다.

1 Backstage란?

Backstage는 밴드 멤버들이 악보를 BPM에 맞춰 자동으로 넘기며 함께 합주할 수 있는 웹 서비스입니다.

✨ 주요 기능
• 악보 자동 넘김 — BPM과 마디 수 기반으로 PDF 악보를 자동 스크롤
• 실시간 동기화 — 여러 멤버가 동시에 같은 곡을 재생하면 자동 싱크
• 오선 감지 — PDF에서 악보 줄(시스템)을 자동/수동으로 인식
• Inst 음원 — 반주 MP3를 업로드하여 악보와 함께 재생
• 다양한 뷰 모드 — 1단/2단/세로스크롤 보기 지원

2 시작하기

1 접속하기
PC 또는 태블릿의 브라우저에서 공유받은 URL로 접속합니다. 별도 앱 설치 없이 바로 사용할 수 있습니다.
2 메인 화면
접속하면 등록된 곡들이 앨범 형태로 표시됩니다. 곡을 클릭하면 상세 페이지로 이동합니다.
3 곡 선택 → 파트 선택 → 재생
곡 상세에서 내 파트(악기)를 클릭하면 악보 플레이어가 열립니다. ▶ 버튼으로 재생하세요.
태블릿(iPad)에서 사용하면 악보를 가장 편하게 볼 수 있습니다. 보면대에 태블릿을 올려놓고 사용해보세요!

3 곡 관리

새 곡 추가하기

1 우측 상단 ＋ 새 곡 버튼 클릭
곡 이름을 입력하고 생성합니다.
2 곡 설정하기
생성 후 ⚙️ 설정에서 다음을 입력합니다:
• BPM — 곡의 빠르기 (메트로놈 기준)
• 박자 — 4/4, 3/4, 6/8 등
• 카운트인 — 재생 전 준비 마디 수
• 커버 이미지 — 앨범 표지 (선택)
• Inst 음원 — 반주 MP3 파일 (선택)
곡을 삭제하면 해당 곡의 모든 파트와 악보 데이터가 함께 삭제됩니다. 신중하게 삭제하세요.

4 악보 업로드

1 파트 추가
곡 상세 페이지에서 ➕ 악보 업로드를 클릭합니다. 파트명(예: 보컬, 기타, 드럼)을 입력하고 PDF를 선택합니다.
2 오선 감지 (시스템 편집)
업로드 후 파트를 클릭 → 오선 편집기가 열립니다.
• 자동 감지: 기본 패턴으로 줄을 나눕니다
• 수동 조정: 각 줄의 위치와 마디 수를 직접 설정
• 줄마다 도돌이표, 볼타 설정도 가능합니다
3 마디 수 설정
각 줄(시스템)에 포함된 마디 수를 정확히 입력해야 자동 넘김이 정확해집니다. BPM과 박자를 기반으로 시간이 자동 계산됩니다.
오선 감지가 잘 안 되면 수동 패턴을 사용하세요. 예: 4,4,4,4 (4줄, 각 4마디)

5 플레이어 사용법

기본 조작

• ▶ 재생 — 카운트인 후 BPM에 맞춰 악보 자동 넘김 시작
• ⏸ 일시정지 — 현재 위치에서 정지
• ⏹ 정지 — 처음으로 되돌아가기
• 시크바 — 드래그하여 원하는 위치로 이동
• BPM ± — 실시간 템포 조절

보기 모드

• 1단 보기 — 한 페이지씩 표시 (기본)
• 2단 보기 — 두 페이지를 나란히 (태블릿 가로 모드 추천)
• 세로 스크롤 — 전체 악보를 위아래로 스크롤

스타일 설정

우측 ⚙️ 설정 패널에서 악보 표시를 커스터마이즈할 수 있습니다:
• 좌/우 여백, 시스템 간격
• 진행선 색상/두께
• 비활성 영역 어둡기
설정은 파트별로 저장됩니다.

6 실시간 합주

여러 멤버가 같은 곡에 접속하면 자동으로 세션이 연결됩니다.

• 한 사람이 ▶ 재생하면 모든 멤버에게 동시에 카운트인 시작
• 일시정지, 정지, 템포 변경도 모두 동기화
• 상단 멤버 배지를 클릭하면 현재 접속자 목록 확인 가능
• Inst 음원도 동기화되어 모두 같은 반주를 들을 수 있습니다

솔로 / 투게더 모드

• Together — 다른 멤버와 동기화 (합주 모드)
• Solo — 혼자 독립적으로 재생 (개인 연습 모드)
합주 시 모든 멤버가 "Together" 모드인지 확인하세요. Solo 모드에서는 동기화가 안 됩니다.

7 FAQ

악보가 BPM과 안 맞아요
오선 편집기에서 각 줄의 마디 수가 정확한지 확인하세요. 마디 수가 틀리면 넘기는 타이밍이 어긋납니다.
PDF가 업로드 안 돼요
PDF 파일만 지원됩니다. 이미지 파일(JPG, PNG)은 먼저 PDF로 변환 후 업로드하세요.
태블릿에서 화면이 잘려요
브라우저의 전체 화면 모드를 사용해보세요. 또는 보기 모드를 변경해보세요.
다른 멤버와 싱크가 안 맞아요
네트워크 상태에 따라 약간의 지연이 있을 수 있습니다. 모두 같은 Wi-Fi에 접속하면 지연이 줄어듭니다.
Inst 음원이 재생 안 돼요
브라우저 정책상 화면을 한 번 터치해야 오디오가 활성화됩니다. 재생 전에 화면을 터치하세요.
도돌이표가 있는 곡은 어떻게 하나요?
오선 편집기에서 각 줄의 반복 시작/끝과 볼타 설정을 사용하세요. 재생 순서 편집도 가능합니다.
